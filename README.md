# Dag 1 –  Onepager

## Waar gaat deze site over?
Deze website is mijn persoonlijke onepager.  
De focus ligt op **interactie, micro-animations en een rustige dark-mode uitstraling**.  
Ik experimenteer met subtiele details zoals een custom cursor, hover-effecten en een vaste muziekspeler om de site meer karakter te geven, zonder dat het druk wordt.

---

## Wat heb ik gedaan?
Vandaag heb ik de basis van mijn onepager opgezet.  
Ik heb gewerkt aan:
- De algemene layout en structuur (semantische HTML)
- Een dark-mode design met vaste design tokens (kleuren & fonts)
- Een custom cursor met meerdere lagen en easing
- Hover-interacties op tekst en elementen
- Een vaste, inklapbare Spotify-muziekspeler aan de zijkant
- Micro-animations (hover feedback, easing, schaal- en positie-effecten)

De focus lag vooral op **gevoel en interactie**, niet alleen op hoe iets eruitziet maar ook hoe het aanvoelt.

---

## Hoe lang duurde het?
Ongeveer **6–7 uur**.

Een groot deel van de tijd ging naar:
- Experimenteren met animaties en easing
- Fine-tunen van hover feedback
- Structuur aanbrengen zodat de CSS overzichtelijk en herbruikbaar blijft

---

## Wat heb ik geleerd?
- Hoe belangrijk het is om te werken met **design tokens** in plaats van losse kleuren
- Dat micro-interactions (kleine bewegingen, easing) veel invloed hebben op de beleving
- Hoe je semantische HTML (`aside`, `details`, `summary`) kunt gebruiken voor interactieve UI zonder veel JavaScript
- Dat minder classes vaak zorgt voor **duidelijkere en schonere CSS**
- Hoe kleine animaties “hard” of “zacht” kunnen aanvoelen afhankelijk van easing en timing

---

## Wat ga ik morgen doen?
- De layout verder verfijnen (uitlijning van tekst en sections)
- De boeken-sectie verder uitwerken
- Responsiveness verbeteren voor mobiel
- Kijken of ik scroll-gedrag (snap of animatie) wil toevoegen
- Kleine visuele details afronden en opruimen in de code
